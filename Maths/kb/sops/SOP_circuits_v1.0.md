# SOP_circuits_v1.0

## 1. Objective
To define the process for managing the electronic circuits simulator/tool.

## 2. Scope
Applies to `Maths/apps/circuits/`.

## 3. Procedure
1. **Adding Components**: To add a new circuit component, define it in the corresponding JavaScript module inside `js/`.
2. **UI Tweaks**: Adjust SVG dimensions and classes in `index.html`.
3. **Testing**: Test the interaction of drag-and-drop components if applicable.

## 4. Troubleshooting
- **Drag-and-Drop issues**: Ensure event listeners are correctly bound after DOM load.

## 5. Revision History
- **v1.0**: Initial Creation.
