# SOP_automatique_v1.0

## 1. Objective
To document the maintenance and update procedures for the `automatique` application within the GEII Visual Toolbox.

## 2. Scope
Applies only to `Maths/apps/automatique/`.

## 3. Procedure
1. **Component Updates**: Locate specific logic within the `apps/automatique/js/` directory.
2. **UI Changes**: Modify `apps/automatique/index.html`.
3. **Integration**: Ensure the app functions when loaded via the global galaxy UI.

## 4. Troubleshooting
- If automatic logic fails to render, check the console for syntax errors in the module imports.

## 5. Revision History
- **v1.0**: Initial Creation.
