# KB03050001: Maths Calculator

## 1. Feature Description
A built-in calculator for evaluating mathematical expressions and visualizing functions.

## 2. How-To Use
1. Launch the **Maths** application.
2. In the input field, type your mathematical function (e.g., `sin(x) * cos(x)`).
3. Press **Enter** or click the **Calculate/Plot** button.

## 3. Expected Outcome
The app will display the evaluated result or plot the 2D graph of the given function.
