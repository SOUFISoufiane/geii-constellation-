# KB05050001: Sine Wave Generation

## 1. Feature Description
Generate and visualize sine waves with adjustable frequency, amplitude, and phase.

## 2. How-To Use
1. Launch the **Signal Observatory**.
2. Go to the "Generators" tab.
3. Select "Sine Wave".
4. Use the sliders to adjust:
   - **Amplitude**: The height of the wave.
   - **Frequency**: The number of cycles per second.
   - **Phase**: The horizontal shift of the wave.
5. The real-time chart will update immediately.

## 3. Expected Outcome
A smooth, continuous sine wave reflecting your chosen parameters will scroll across the oscilloscope view.
