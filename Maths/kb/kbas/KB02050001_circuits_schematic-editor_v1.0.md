# KB02050001: Circuits Schematic Editor

## 1. Feature Description
This feature allows users to construct electronic circuit schematics by dragging and dropping components.

## 2. How-To Use
1. From the home screen, open the **Circuits** app.
2. Look at the component sidebar on the left.
3. **Drag** a resistor, capacitor, or voltage source onto the main canvas.
4. **Click and drag** from the terminal of one component to another to create a wire connection.
5. Click "Simulate" (if available) to see the voltage flow.

## 3. Expected Outcome
The circuit components will snap into place and wiring will be established, forming a closed or open loop.

## 4. Troubleshooting
- If components don't connect, ensure you are dragging from the exact terminal point.
