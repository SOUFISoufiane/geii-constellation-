# KB01050001: Automatique Overview

## 1. Feature Description
The Automatique app is designed to help you visualize and analyze automatic control systems.

## 2. How-To Use
1. Open the GEII Visual Toolbox from the main Galaxy screen.
2. Click on the **Automatique** module.
3. Input your system parameters.
4. View the generated plots and control responses.

## 3. Expected Outcome
You will see a real-time visualization of the automatic control system response.
